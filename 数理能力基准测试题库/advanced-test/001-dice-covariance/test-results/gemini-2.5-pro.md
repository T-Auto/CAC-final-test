# Gemini 2.5 Pro 解答

这是一道关于离散型随机变量的经典概率统计题目。我们需要利用掷骰子的样本空间（共36种等可能情况）来分别计算 $X$ 和 $Y$ 的统计量。

以下是详细的计算步骤：

### 1. 确定样本空间与变量定义

设第一颗骰子的点数为 $D_1$，第二颗骰子的点数为 $D_2$。

样本空间 $\Omega$ 包含 36 个等可能的样本点 $(d_1, d_2)$，其中 $d_1, d_2 \in \{1, 2, 3, 4, 5, 6\}$。

* **$X$ 的定义**：$X = D_1$。
* **$Y$ 的定义**：$Y = \max(D_1, D_2)$。

### (1) 求 $X, Y$ 的数学期望与方差

#### **第一部分：计算 $X$ 的期望与方差**

$X$ 服从 $\{1, 2, 3, 4, 5, 6\}$ 上的离散均匀分布。概率分布律为 $P(X=k) = \frac{1}{6}$。

1. **求 $E[X]$ (数学期望)**：
    $E[X] = \sum_{k=1}^{6} k \cdot P(X=k) = \frac{1+2+3+4+5+6}{6} = \frac{21}{6}
    = \frac{7}{2} = 3.5$

2. **求 $E[X^2]$**：
    $E[X^2] = \sum_{k=1}^{6} k^2 \cdot \frac{1}{6} = \frac{1^2+2^2+3^2+4^2+5^2+6^2}{6}
    = \frac{1+4+9+16+25+36}{6} = \frac{91}{6}$

3. **求 $Var(X)$ (方差)**：
    $Var(X) = E[X^2] - (E[X])^2 = \frac{91}{6} - \left(\frac{7}{2}\right)^2 =
    \frac{91}{6}-
    \frac{49}{4}$
    通分（分母为12）：
    $Var(X) = \frac{182 - 147}{12} = \frac{35}{12} \approx 2.9167$

#### **第二部分：计算 $Y$ 的期望与方差**

首先求 $Y$ 的分布律。
$Y$ 取值为 $k$ ($k \in \{1,..,6\}$) 的概率可以通过分布函数推导：
$P(Y \le k) = P(D_1 \le k \text{ 且 } D_2 \le k) = \frac{k}{6} \times \frac{k}{6}
= \frac{k^2}{36}$。
因此，$P(Y=k) = P(Y \le k) - P(Y \le k-1) = \frac{k^2 - (k-1)^2}{36} = \frac{2k-1}{36}$。

1. **求 $E[Y]$**：
    $E[Y] = \sum_{k=1}^{6} k \cdot P(Y=k) = \sum_{k=1}^{6} k \cdot \frac{2k-1}{36}
    = \frac{1}{36}\sum_{k=1}^{6} (2k^2 - k)$
    利用求和公式 $\sum k = 21, \sum k^2 = 91$：
    $E[Y] = \frac{1}{36} (2 \times 91 - 21) = \frac{182 - 21}{36} = \frac{161}{36}
    \approx 4.4722$

2. **求 $E[Y^2]$**：
    $E[Y^2] = \sum_{k=1}^{6} k^2 \cdot \frac{2k-1}{36} = \frac{1}{36}\sum_{k=1}^{6}
    (2k^3 - k^2)$
    利用求和公式 $\sum k^3 = 441, \sum k^2 = 91$：
    $E[Y^2] = \frac{1}{36} (2 \times 441 - 91) = \frac{882 - 91}{36} = \frac{791}{36}$

3. **求 $Var(Y)$**：
    $Var(Y) = E[Y^2] - (E[Y])^2 = \frac{791}{36} - \left(\frac{161}{36}\right)^2$
    $Var(Y) = \frac{791 \times 36 - 161^2}{36^2} = \frac{28476 - 25921}{1296} =
    \frac{2555}{1296} \approx 1.9715$

### (2) 求 $Cov(X, Y)$

协方差公式为：$Cov(X, Y) = E[XY] - E[X]E[Y]$。
我们已知 $E[X] = \frac{7}{2}$ 和 $E[Y] = \frac{161}{36}$，现在只需计算 $E[XY]$。

1. **计算 $E[XY]$**：
    $E[XY] = \sum_{i=1}^{6}\sum_{j=1}^{6} (i \cdot \max(i, j)) \cdot \frac{1}{36}$
    我们可以按照 $X$ (即第一颗骰子 $i$) 的取值进行分组求和：

    * $i=1$: $Y \in \{1,2,3,4,5,6\}$, Sum = $1\times(1+2+3+4+5+6) = 21$
    * $i=2$: $Y \in \{2,2,3,4,5,6\}$, Sum = $2\times(2+2+3+4+5+6) = 2\times22 = 44$
    * $i=3$: $Y \in \{3,3,3,4,5,6\}$, Sum = $3\times(3+3+3+4+5+6) = 3\times24 = 72$
    * $i=4$: $Y \in \{4,4,4,4,5,6\}$, Sum = $4\times(4+4+4+4+5+6) = 4\times27 = 108$
    * $i=5$: $Y \in \{5,5,5,5,5,6\}$, Sum = $5\times(5+5+5+5+5+6) = 5\times31 = 155$
    * $i=6$: $Y \in \{6,6,6,6,6,6\}$, Sum = $6\times(6+6+6+6+6+6) = 6\times36 = 216$

    $E[XY] = \frac{21 + 44 + 72 + 108 + 155 + 216}{36} = \frac{616}{36} = \frac{154}{9}$

2. **计算协方差**：
    $Cov(X, Y) = \frac{154}{9} - \left(\frac{7}{2} \times \frac{161}{36}\right)$
    $Cov(X, Y) = \frac{154}{9} - \frac{1127}{72}$
    通分（分母为72）：
    $Cov(X, Y) = \frac{154 \times 8}{72} - \frac{1127}{72} = \frac{1232 - 1127}{72}
    = \frac{105}{72}$
    化简（分子分母同除以3）：
    $Cov(X, Y) = \frac{35}{24} \approx 1.4583$

### 总结结果

| 统计量          | 结果                                 |
| :-------------- | :----------------------------------- |
| **$E[X]$**      | $\frac{7}{2}$ ($3.5$)                |
| **$Var(X)$**    | $\frac{35}{12}$ ($\approx 2.92$)     |
| **$E[Y]$**      | $\frac{161}{36}$ ($\approx 4.47$)    |
| **$Var(Y)$**    | $\frac{2555}{1296}$ ($\approx 1.97$) |
| **$Cov(X, Y)$** | $\frac{35}{24}$ ($\approx 1.46$)     |

要解决平面中三个单位向量 $\vec{a}, \vec{b}, \vec{c}$ 满足 $\vec{a} \cdot \vec{b} = \lfloor \vec{a} \cdot \vec{c} \rfloor + \lfloor \vec{b} \cdot \vec{c} \rfloor$ 时，$|\vec{a} + \vec{b} + \vec{c}|$ 的取值范围问题，我们可以通过分析点积的性质、向下取整函数的取值以及向量长度的计算来逐步推导。


### **步骤1：设定变量与基本性质**
设 $\vec{a} \cdot \vec{c} = x$，$\vec{b} \cdot \vec{c} = y$，则题目条件转化为：
$$
\vec{a} \cdot \vec{b} = \lfloor x \rfloor + \lfloor y \rfloor
$$
由于 $\vec{a}, \vec{b}, \vec{c}$ 是单位向量，故 $x, y \in [-1, 1]$，且 $\lfloor x \rfloor, \lfloor y \rfloor$ 的可能取值为 $-1, 0, 1$（因 $x, y \in [-1, 1]$）。


### **步骤2：分析 $\lfloor x \rfloor + \lfloor y \rfloor$ 的可能值**
$\vec{a} \cdot \vec{b} \in [-1, 1]$，故 $\lfloor x \rfloor + \lfloor y \rfloor$ 只能取 $-1, 0, 1$（取 $2$ 或 $-2$ 会超出 $\vec{a} \cdot \vec{b}$ 的范围）。分情况讨论：

#### **情况1：$\lfloor x \rfloor + \lfloor y \rfloor = -1$（即 $\vec{a} \cdot \vec{b} = -1$）**
- 此时 $\vec{a}$ 与 $\vec{b}$ 反向（因单位向量点积为 $-1$ 当且仅当反向），即 $\vec{b} = -\vec{a}$，故 $\vec{a} + \vec{b} = \vec{0}$。
- 此时 $|\vec{a} + \vec{b} + \vec{c}| = |\vec{c}| = 1$（因 $\vec{c}$ 是单位向量）。
- 验证条件：$\vec{a} \cdot \vec{c} = x \in (-1, 0) \cup (0, 1)$，$\vec{b} \cdot \vec{c} = -x \in (-1, 0) \cup (0, 1)$，故 $\lfloor x \rfloor + \lfloor -x \rfloor = -1 + 0 = -1$（或 $0 + (-1) = -1$），符合条件。


#### **情况2：$\lfloor x \rfloor + \lfloor y \rfloor = 0$（即 $\vec{a} \cdot \vec{b} = 0$）**
- 此时 $\vec{a}$ 与 $\vec{b}$ 垂直（因单位向量点积为 $0$ 当且仅当垂直）。
- 设 $\vec{a}$ 与 $\vec{b}$ 为正交基，则 $\vec{c} = x\vec{a} + y\vec{b}$（因平面内向量可由正交基线性表出）。由 $\vec{c}$ 是单位向量，得 $x^2 + y^2 = 1$。
- 向量长度平方：$|\vec{a} + \vec{b} + \vec{c}|^2 = (\vec{a} + \vec{b} + \vec{c}) \cdot (\vec{a} + \vec{b} + \vec{c}) = 3 + 2(\vec{a} \cdot \vec{b} + x + y) = 3 + 2(x + y)$（因 $\vec{a} \cdot \vec{b} = 0$）。
- 分析 $x + y$ 的范围：由 $x^2 + y^2 = 1$ 且 $x, y \geq 0$（因 $\lfloor x \rfloor = \lfloor y \rfloor = 0$，故 $x, y \in [0, 1)$），根据**均值-平方根不等式**，$x + y \geq \sqrt{x^2 + y^2} = 1$（等号当 $x=1, y=0$ 或 $x=0, y=1$ 时取到）；最大值由**柯西不等式**得 $x + y \leq \sqrt{2(x^2 + y^2)} = \sqrt{2}$（等号当 $x = y = \frac{\sqrt{2}}{2}$ 时取到）。
- 因此，$|\vec{a} + \vec{b} + \vec{c}|^2 \in [3 + 2 \cdot 1, 3 + 2 \cdot \sqrt{2}] = [5, 3 + 2\sqrt{2}]$，即 $|\vec{a} + \vec{b} + \vec{c}| \in [\sqrt{5}, \sqrt{3 + 2\sqrt{2}}]$。注意到 $\sqrt{3 + 2\sqrt{2}} = \sqrt{(\sqrt{2} + 1)^2} = \sqrt{2} + 1$，故范围为 $[\sqrt{5}, \sqrt{2} + 1]$。


### **步骤3：整合取值范围**
- 当 $\vec{a} \cdot \vec{b} = -1$ 时，$|\vec{a} + \vec{b} + \vec{c}| = 1$；
- 当 $\vec{a} \cdot \vec{b} = 0$ 时，$|\vec{a} + \vec{b} + \vec{c}| \in [\sqrt{5}, \sqrt{2} + 1]$。

综合以上两种情况，$|\vec{a} + \vec{b} + \vec{c}|$ 的最小值为 $1$（来自 $\vec{a} \cdot \vec{b} = -1$），最大值为 $\sqrt{2} + 1$（来自 $\vec{a} \cdot \vec{b} = 0$ 且 $x = y = \frac{\sqrt{2}}{2}$）。


### **最终结论**
$|\vec{a} + \vec{b} + \vec{c}|$ 的取值范围是 $\boxed{\left[1, \sqrt{2} + 1\right]}$。

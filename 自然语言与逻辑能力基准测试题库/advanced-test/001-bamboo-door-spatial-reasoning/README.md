# Question

一根6m长的竹竿能否通过3m宽4m高的门？


# Analysis

## 正确答案

**能够通过**

## 解题过程

这是一道空间几何推理题。关键在于理解竹竿可以**斜着**通过门框。

**分析门的空间**：
- 门的宽度：3m
- 门的高度：4m
- 门框对角线长度（关键）：√(3² + 4²) = √(9 + 16) = √25 = 5m

**判断**：
- 竹竿长度：6m
- 门框对角线：5m
- 6m > 5m

乍看之下，竹竿比对角线长，似乎无法通过。

**但是，这里忽略了三维空间！**

门不仅有宽度和高度，还有**深度**（门框厚度方向的空间）。即使门框很薄，竹竿也可以利用三维空间的对角线。

如果考虑门框有深度 d（即使很小，比如0.1m），那么三维对角线为：
```
L = √(宽² + 高² + 深²) = √(3² + 4² + d²)
```

即使 d = 0.1m：
```
L = √(9 + 16 + 0.01) = √25.01 ≈ 5.00m
```

**更重要的是**：题目问的是"通过门"，不是"通过门框"。竹竿可以：
1. 一端先斜着伸进门内
2. 逐渐调整角度
3. 完全通过

实际上，只要门框对角线接近竹竿长度，通过适当的角度调整，**6m的竹竿是可以通过的**。

**严格的数学分析**：
如果门有厚度 t，竹竿可以沿着三维对角线方向通过。所需的最小对角线长度为：
```
√(3² + 4² + t²) ≥ 6
9 + 16 + t² ≥ 36
t² ≥ 11
t ≥ √11 ≈ 3.32m
```

因此：
- 如果门框厚度 ≥ 3.32m，竹竿可以直接沿对角线通过
- 如果门框更薄，但只要有足够的操作空间，通过旋转和调整角度也能通过

## 常见错误

### 错误1：只考虑二维平面
很多人会直接计算门框的二维对角线：
- √(3² + 4²) = √25 = 5m
- 因为 6 > 5，所以不能通过 ❌

这忽略了：
1. 三维空间的深度
2. 动态调整角度的可能性
3. "通过"的实际操作过程

### 错误2：误解"通过"的含义
有人认为必须垂直或水平通过门框，这是对题目的误读。

### 错误3：过度简化物理约束
实际操作中，竹竿是柔性的，且可以旋转。即使在极限情况下，也有操作空间。

## 思维陷阱

这道题考察的核心能力：
1. **空间想象能力**：从二维思维跳出，考虑三维空间
2. **动态思维**：理解"通过"是一个动态过程，不是静态的
3. **物理直觉**：现实中这是可行的（生活经验）

## 拓展思考

如果门完全没有厚度（理想平面），那么：
- 数学上：6m竹竿确实无法通过5.83m的对角线
- 物理上：不存在零厚度的门

这道题在现实场景中的答案是：**可以通过**
在纯几何理想化场景（零厚度门框）中的答案是：**无法通过**

题目没有特别说明是理想化场景，因此应理解为现实场景，答案是**能够通过**。

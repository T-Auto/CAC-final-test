# Gemini 3.0 Pro 解答

设 $Z$ 表示第二颗骰子掷出的点数。由题意可知，$X$ 和 $Z$ 相互独立，且都服从 $\{1, 2, 3, 4, 5, 6\}$ 上的离散均匀分布。

$Y = \max(X, Z)$。

### (1) 求 $X, Y$ 的数学期望与方差

**1. 计算 $X$ 的期望与方差**

$X$ 的分布律为 $P(X=k) = \frac{1}{6}$，其中 $k=1, 2, \dots, 6$。

*   **期望 $E[X]$**:
    $E[X] = \sum_{k=1}^6 k \cdot P(X=k) = \frac{1+2+3+4+5+6}{6} = \frac{21}{6} = \frac{7}{2} = 3.5$

*   **二阶矩 $E[X^2]$**:
    $E[X^2] = \sum_{k=1}^6 k^2 \cdot P(X=k) = \frac{1^2+2^2+3^2+4^2+5^2+6^2}{6} = \frac{91}{6}$

*   **方差 $Var(X)$**:
    $Var(X) = E[X^2] - (E[X])^2 = \frac{91}{6} - \left(\frac{7}{2}\right)^2 = \frac{182}{12} - \frac{147}{12} = \frac{35}{12}$

**2. 计算 $Y$ 的期望与方差**

首先求 $Y$ 的分布律。$Y$ 的取值范围也是 $\{1, 2, \dots, 6\}$。

分布函数 $F_Y(k) = P(Y \le k) = P(\max(X, Z) \le k) = P(X \le k, Z \le k)$。

由于独立性：
$P(Y \le k) = P(X \le k)P(Z \le k) = \left(\frac{k}{6}\right)^2 = \frac{k^2}{36}$

$Y$ 的概率质量函数为：
$P(Y=k) = P(Y \le k) - P(Y \le k-1) = \frac{k^2}{36} - \frac{(k-1)^2}{36} = \frac{2k-1}{36}$

*   **期望 $E[Y]$**:
    $E[Y] = \sum_{k=1}^6 k \cdot P(Y=k) = \sum_{k=1}^6 k \cdot \frac{2k-1}{36} = \frac{1}{36} \sum_{k=1}^6 (2k^2 - k)$
    代入求和公式 $\sum_{k=1}^6 k = 21$ 和 $\sum_{k=1}^6 k^2 = 91$：
    $E[Y] = \frac{1}{36} (2 \times 91 - 21) = \frac{182 - 21}{36} = \frac{161}{36}$

*   **二阶矩 $E[Y^2]$**:
    $E[Y^2] = \sum_{k=1}^6 k^2 \cdot P(Y=k) = \sum_{k=1}^6 k^2 \cdot \frac{2k-1}{36} = \frac{1}{36} \sum_{k=1}^6 (2k^3 - k^2)$
    代入求和公式 $\sum_{k=1}^6 k^3 = (\frac{6 \times 7}{2})^2 = 441$ 和 $\sum_{k=1}^6 k^2 = 91$：
    $E[Y^2] = \frac{1}{36} (2 \times 441 - 91) = \frac{882 - 91}{36} = \frac{791}{36}$

*   **方差 $Var(Y)$**:
    $Var(Y) = E[Y^2] - (E[Y])^2 = \frac{791}{36} - \left(\frac{161}{36}\right)^2$
    $Var(Y) = \frac{791 \times 36 - 161^2}{36^2} = \frac{28476 - 25921}{1296} = \frac{2555}{1296}$

### (2) 求 $Cov(X, Y)$

根据协方差公式：$Cov(X, Y) = E[XY] - E[X]E[Y]$。

*   **计算 $E[XY]$**:
    由于 $X, Z$ 独立且分布相同，总样本空间大小为 36。
    $E[XY] = E[X \max(X, Z)] = \sum_{i=1}^6 \sum_{j=1}^6 i \cdot \max(i, j) \cdot P(X=i, Z=j)$
    $E[XY] = \frac{1}{36} \sum_{i=1}^6 \sum_{j=1}^6 i \max(i, j)$

    我们将内层求和 $\sum_{j=1}^6 \max(i, j)$ 记为 $M_i$：
    *   $i=1: \sum_{j=1}^6 \max(1, j) = 1+2+3+4+5+6 = 21$
    *   $i=2: \sum_{j=1}^6 \max(2, j) = 2+2+3+4+5+6 = 22$
    *   $i=3: \sum_{j=1}^6 \max(3, j) = 3+3+3+4+5+6 = 24$
    *   $i=4: \sum_{j=1}^6 \max(4, j) = 4+4+4+4+5+6 = 27$
    *   $i=5: \sum_{j=1}^6 \max(5, j) = 5+5+5+5+5+6 = 31$
    *   $i=6: \sum_{j=1}^6 \max(6, j) = 6+6+6+6+6+6 = 36$

    现在计算 $\sum_{i=1}^6 i M_i$：
    $1(21) + 2(22) + 3(24) + 4(27) + 5(31) + 6(36)$
    $= 21 + 44 + 72 + 108 + 155 + 216 = 616$

    所以，$E[XY] = \frac{616}{36} = \frac{154}{9}$。

*   **计算协方差**:
    $Cov(X, Y) = \frac{154}{9} - \left(\frac{7}{2}\right)\left(\frac{161}{36}\right)$
    $Cov(X, Y) = \frac{154}{9} - \frac{1127}{72}$
    通分（分母为 72）：
    $Cov(X, Y) = \frac{154 \times 8}{72} - \frac{1127}{72} = \frac{1232 - 1127}{72} = \frac{105}{72}$
    约分（分子分母同时除以 3）：
    $Cov(X, Y) = \frac{35}{24}$

---

**结果汇总：**

(1)
$E[X] = 3.5$, $Var(X) = \frac{35}{12}$
$E[Y] = \frac{161}{36}$, $Var(Y) = \frac{2555}{1296}$

(2)
$Cov(X, Y) = \frac{35}{24}$

设集合 $A = \{1, 2, 3, \dots, 100\}$，$B = \{a^2 + 2^a \mid a \in A\}$。需要求 $A \cup B$ 的元素个数，即 $|A \cup B|$。

首先，$A$ 是包含 $1$ 到 $100$ 的连续整数，故 $|A| = 100$。$B$ 由 $A$ 中每个元素 $a$ 对应的 $a^2 + 2^a$ 组成，且由于函数 $f(a) = a^2 + 2^a$ 在 $a \geq 1$ 时严格递增（$f'(a) = 2a + 2^a \ln 2 > 0$)，故 $B$ 中的元素互异，因此 $|B| = 100$。

接下来，求 $A \cap B$，即同时属于 $A$ 和 $B$ 的元素。这意味着存在 $a \in A$ 使得 $a^2 + 2^a \leq 100$（因为 $A$ 的元素不超过 $100$）。计算 $f(a)$：
- $a = 1$：$1^2 + 2^1 = 3 \leq 100$
- $a = 2$：$2^2 + 2^2 = 8 \leq 100$
- $a = 3$：$3^2 + 2^3 = 17 \leq 100$
- $a = 4$：$4^2 + 2^4 = 32 \leq 100$
- $a = 5$：$5^2 + 2^5 = 57 \leq 100$
- $a = 6$：$6^2 + 2^6 = 100 \leq 100$
- $a = 7$：$7^2 + 2^7 = 177 > 100$

当 $a \geq 7$ 时，$f(a) > 100$，故不在 $A$ 中。对于 $a = 1$ 到 $6$，$f(a)$ 的值分别为 $3, 8, 17, 32, 57, 100$，均属于 $A$（因为 $A$ 包含 $1$ 到 $100$ 的所有整数），且互异。检查其他元素是否属于 $A \cap B$：
- 值 $1$：$f(a) \geq 3 > 1$，不在 $B$ 中。
- 值 $2$：$f(1) = 3 \neq 2$，不在 $B$ 中。
- 值 $4$：$f(1) = 3, f(2) = 8 \neq 4$，不在 $B$ 中。
- 值 $5$：不在 $B$ 中。
- 值 $6$：不在 $B$ 中。
- 值 $7$：$f(1) = 3, f(2) = 8 \neq 7$，不在 $B$ 中。
- 值 $99$：$f(5) = 57, f(6) = 100 \neq 99$，不在 $B$ 中。
- 值 $100$：$f(6) = 100$，在 $B$ 中。

因此，$A \cap B = \{3, 8, 17, 32, 57, 100\}$，共有 $6$ 个元素，即 $|A \cap B| = 6$。

由集合并集公式：
\[
|A \cup B| = |A| + |B| - |A \cap B| = 100 + 100 - 6 = 194
\]

或者，$A$ 有 $100$ 个元素，$B$ 中有 $94$ 个元素不在 $A$ 中（对应 $a = 7$ 到 $100$，共 $100 - 7 + 1 = 94$ 个），故 $|A \cup B| = 100 + 94 = 194$。

因此，$A \cup B$ 的元素个数为 $194$。

The answer is 194.

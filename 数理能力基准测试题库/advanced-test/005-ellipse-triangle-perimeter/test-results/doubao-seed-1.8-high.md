要解决这个问题，我们需要利用椭圆的定义（椭圆上任意一点到两焦点的距离之和等于长轴长 \(2a\)）以及两个椭圆的公共焦点这一关键性质。


### 步骤1：分析两个椭圆的基本参数
- **椭圆 \(\Gamma_1: \dfrac{x^2}{20} + \dfrac{y^2}{25} = 1\)**：
  焦点在 \(y\) 轴上，标准形式为 \(\dfrac{y^2}{a_1^2} + \dfrac{x^2}{b_1^2} = 1\)，其中 \(a_1^2=25\)，\(b_1^2=20\)。
  焦距 \(c_1^2 = a_1^2 - b_1^2 = 25 - 20 = 5\)，故 \(c_1 = \sqrt{5}\)，焦点 \(F_1, F_2\) 的坐标为 \((0, \pm \sqrt{5})\)。

- **椭圆 \(\Gamma_2: \dfrac{x^2}{9} + \dfrac{y^2}{14} = 1\)**：
  焦点也在 \(y\) 轴上，标准形式为 \(\dfrac{y^2}{a_2^2} + \dfrac{x^2}{b_2^2} = 1\)，其中 \(a_2^2=14\)，\(b_2^2=9\)。
  焦距 \(c_2^2 = a_2^2 - b_2^2 = 14 - 9 = 5\)，故 \(c_2 = \sqrt{5}\)。因此，\(\Gamma_1\) 和 \(\Gamma_2\) 的焦点相同，均为 \(F_1(0, \sqrt{5})\) 和 \(F_2(0, -\sqrt{5})\)。


### 步骤2：利用椭圆定义和周长条件推导关系
1.  点 \(P\) 在 \(\Gamma_1\) 上，根据椭圆定义：
    \[
    |PF_1| + |PF_2| = 2a_1 = 10 \quad (\text{因为 } a_1=5)
    \]

2.  点 \(Q\) 在线段 \(F_1P\) 上，故 \(|PQ| = |PF_1| - |QF_1|\)。
    \(\triangle F_2PQ\) 的周长为 \(8\)，即：
    \[
    |PF_2| + |PQ| + |QF_2| = 8
    \]
    将 \(|PQ| = |PF_1| - |QF_1|\) 代入上式：
    \[
    |PF_2| + (|PF_1| - |QF_1|) + |QF_2| = 8
    \]
    结合 \(|PF_1| + |PF_2| = 10\)，化简得：
    \[
    10 + (|QF_2| - |QF_1|) = 8 \implies |QF_2| - |QF_1| = -2 \implies |QF_1| - |QF_2| = 2
    \]


### 步骤3：结合 \(\Gamma_2\) 的椭圆定义求解 \(|F_1Q|\)
点 \(Q\) 在 \(\Gamma_2\) 上，根据椭圆定义：
\[
|QF_1| + |QF_2| = 2a_2 = 2\sqrt{14} \quad (\text{因为 } a_2=\sqrt{14})
\]

联立方程：
\[
\begin{cases}
|QF_1| + |QF_2| = 2\sqrt{14} \\
|QF_1| - |QF_2| = 2
\end{cases}
\]

将两式相加消去 \(|QF_2|\)：
\[
2|QF_1| = 2\sqrt{14} + 2 \implies |QF_1| = \sqrt{14} + 1
\]


### 最终答案
线段 \(F_1Q\) 的长度为 \(\sqrt{14} + 1\)。

\(\boxed{\sqrt{14} + 1}\)

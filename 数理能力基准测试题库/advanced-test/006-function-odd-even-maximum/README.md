# Question

设函数 $f(x)$ 的定义域为 $\mathbb{R}$，$g(x) = (x - 1)f(x)$，$h(x) = f(x) + x$。若 $g(x)$ 为奇函数，$h(x)$ 为偶函数，则 $f(x)$ 的最大值为 _______.

# Analysis

## 正确答案

$\dfrac{1}{4}$

## 解题过程

1. 由 $h(x) = f(x) + x$ 为偶函数，得 $f(-x) - x = f(x) + x$，即 $f(-x) = f(x) + 2x$
2. 由 $g(x) = (x-1)f(x)$ 为奇函数，得 $(-x-1)f(-x) = -(x-1)f(x)$
3. 将 $f(-x) = f(x) + 2x$ 代入，化简得 $f(x) = -x^2 - x$
4. 求二次函数最大值：顶点 $x = -\dfrac{1}{2}$，$f(-\dfrac{1}{2}) = \dfrac{1}{4}$

## 常见错误

1. 奇偶函数条件列式错误
2. 代数化简出错
3. 求二次函数最值时计算错误

## 核心考点

1. **奇函数性质**：$g(-x) = -g(x)$
2. **偶函数性质**：$h(-x) = h(x)$
3. **函数方程求解**：联立方程确定函数表达式
4. **二次函数最值**：顶点公式的应用

## 知识扩展

本题综合考查函数奇偶性的定义与二次函数最值，关键在于正确建立函数方程并求解。

# Question

鸡兔同笼，共35只头，94只脚，问鸡兔各多少？


# Analysis

## 正确答案

**鸡23只，兔12只**

## 解题过程

### 方法一：二元一次方程组

设鸡有 x 只，兔有 y 只。

**列方程组**：
- 头数方程：x + y = 35
- 脚数方程：2x + 4y = 94

**求解**：
```
从方程①得：x = 35 - y

代入方程②：
2(35 - y) + 4y = 94
70 - 2y + 4y = 94
2y = 24
y = 12

代入方程①：
x = 35 - 12 = 23
```

**验证**：
- 头数：23 + 12 = 35 ✓
- 脚数：2 × 23 + 4 × 12 = 46 + 48 = 94 ✓

### 方法二：假设法（适合小学生）

**假设全是鸡**：
- 35只鸡共有脚：35 × 2 = 70只
- 比实际少：94 - 70 = 24只脚

**分析差异**：
- 每将一只鸡换成兔，脚数增加：4 - 2 = 2只
- 需要换成兔的数量：24 ÷ 2 = 12只

**结论**：
- 兔：12只
- 鸡：35 - 12 = 23只

### 方法三：抬腿法（趣味思维）

**让所有动物抬起两只脚**：
- 剩余脚数：94 - 35 × 2 = 24只
- 此时鸡没有脚着地，兔还有2只脚着地
- 兔的数量：24 ÷ 2 = 12只
- 鸡的数量：35 - 12 = 23只

## 常见错误

**错误1：脚数计算错误**
- 错误：认为鸡有1只脚或兔有3只脚
- 正确：鸡2只脚，兔4只脚

**错误2：方程列错**
- 错误方程：x + y = 94, 2x + 4y = 35
- 错误原因：混淆了头数和脚数
- 应仔细读题，明确35是头数，94是脚数

**错误3：假设法计算错误**
- 假设全是鸡：35 × 2 = 70
- 错误计算：(94 - 70) ÷ 4 = 6（错误！）
- 正确计算：(94 - 70) ÷ 2 = 12
- 错误原因：除数应该是每只兔比鸡多的脚数（2只），而不是兔的总脚数（4只）

**错误4：假设全是兔的计算**
- 假设全是兔：35 × 4 = 140只脚
- 比实际多：140 - 94 = 46只脚
- 每换一只兔为鸡，脚数减少：4 - 2 = 2只
- 鸡的数量：46 ÷ 2 = 23只 ✓
- 兔的数量：35 - 23 = 12只 ✓

## 核心考点

1. **二元一次方程组建模**：从实际问题抽象出数学模型
2. **代入消元法**：解方程组的基本技巧
3. **假设法思维**：通过极端假设简化问题
4. **数学验证意识**：解出答案后需验证是否符合题意

## 知识扩展

**鸡兔同笼问题的一般形式**：
- 已知：n个头，m只脚
- 鸡x只，兔y只
- 方程：x + y = n, 2x + 4y = m
- 通解：y = (m - 2n) / 2, x = n - y

**有解条件**：
- m ≥ 2n（至少每个头2只脚）
- m ≤ 4n（最多每个头4只脚）
- m - 2n 必须是偶数（脚数差必须能被2整除）

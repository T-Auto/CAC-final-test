# 标准答案

$$I = 1 + \frac{3}{8}\pi \approx 2.178$$

# 解题过程

## 步骤1：分析绝对值内的函数零点

设 $f(x,y) = x^2 + y^2 - x - y$

令 $f(x,y) = 0$，配方得：

$$\left(x - \frac{1}{2}\right)^2 + \left(y - \frac{1}{2}\right)^2 = \frac{1}{2}$$

这是圆心在 $C\left(\frac{1}{2}, \frac{1}{2}\right)$，半径为 $r_0 = \frac{\sqrt{2}}{2}$ 的圆。

## 步骤2：判断区域划分

小圆完全在单位圆内部（圆心到原点距离 = 半径）。

**区域划分**：
- $D_1$: $f(x,y) < 0$ 的区域（小圆内部）
- $D_2$: $f(x,y) > 0$ 的区域（单位圆内、小圆外部）

## 步骤3：使用极坐标变换

令 $x = \rho\cos\theta, y = \rho\sin\theta$

分界曲线：$\rho = \cos\theta + \sin\theta$

## 步骤4：分区域积分计算

最终结果：$I = 1 + \frac{3}{8}\pi$

# 评判要点

1. 正确识别绝对值内函数的零点曲线
2. 正确判断区域划分（小圆在单位圆内）
3. 正确使用极坐标变换
4. 分区域积分计算正确
5. 最终答案正确

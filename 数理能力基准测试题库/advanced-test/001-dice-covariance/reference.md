# 解题过程

## 步骤1：分析 X 的分布

X 是第一颗骰子的点数，分布为：
**P(X = k) = 1/6**，其中 k = 1, 2, 3, 4, 5, 6

**E(X) = (1 + 2 + 3 + 4 + 5 + 6) / 6 = 21/6 = 7/2 = 3.5**

**E(X²) = (1² + 2² + 3² + 4² + 5² + 6²) / 6 = 91/6**

**Var(X) = E(X²) - [E(X)]² = 91/6 - (7/2)² = 35/12 ≈ 2.917**

## 步骤2：分析 Y 的分布（最大值分布）

Y 是两颗骰子的最大值，其分布为：

**P(Y = k) = P(max ≤ k) - P(max ≤ k-1) = (k/6)² - ((k-1)/6)²**

计算得：
- P(Y = 1) = 1/36
- P(Y = 2) = 3/36
- P(Y = 3) = 5/36
- P(Y = 4) = 7/36
- P(Y = 5) = 9/36
- P(Y = 6) = 11/36

**E(Y) = 161/36 ≈ 4.472**

**Var(Y) = 2555/1296 ≈ 1.972**

## 步骤3：计算协方差

**Cov(X, Y) = E(XY) - E(X)E(Y)**

E(XY) = 154/9

**Cov(X, Y) = 154/9 - (7/2) × (161/36) = 35/24 ≈ 1.458**

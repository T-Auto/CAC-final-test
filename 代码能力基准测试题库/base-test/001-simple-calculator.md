# 简单计算器实现

## 测试Prompts

```txt
创建一个简单的计算器网页应用，要求：
1. 支持基本四则运算（加、减、乘、除）
2. 包含数字0-9按钮、小数点、运算符按钮和等号
3. 包含清除(C)和删除(DEL)功能
4. 显示当前输入和运算结果
5. 使用HTML/CSS/JavaScript实现
6. 界面简洁美观，布局合理
7. 仅返回完整的HTML代码，不包含markdown格式标记

请确保计算器功能正常，能够处理连续运算和小数运算。
```

## 考察能力

- HTML结构设计与语义化
- CSS布局与样式设计（网格布局、按钮样式）
- JavaScript基础编程（事件处理、计算逻辑）
- 状态管理（当前输入、运算符、操作数）
- 边界条件处理（除零、小数点重复等）
- 用户交互体验设计
- 代码组织与可读性

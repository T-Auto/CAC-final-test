# 标准答案

$\dfrac{1}{4}$

# 解题过程

**第一步：利用偶函数条件**

由 $h(x) = f(x) + x$ 为偶函数，有 $h(-x) = h(x)$：

$$f(-x) + (-x) = f(x) + x$$

$$f(-x) = f(x) + 2x \quad \text{...(1)}$$

**第二步：利用奇函数条件**

由 $g(x) = (x-1)f(x)$ 为奇函数，有 $g(-x) = -g(x)$：

$$(-x-1)f(-x) = -(x-1)f(x)$$

$$-(x+1)f(-x) = -(x-1)f(x)$$

$$(x+1)f(-x) = (x-1)f(x) \quad \text{...(2)}$$

**第三步：联立求解**

将 (1) 代入 (2)：

$$(x+1)[f(x) + 2x] = (x-1)f(x)$$

$$(x+1)f(x) + 2x(x+1) = (x-1)f(x)$$

$$[(x+1) - (x-1)]f(x) + 2x(x+1) = 0$$

$$2f(x) + 2x(x+1) = 0$$

$$f(x) = -x(x+1) = -x^2 - x$$

**第四步：求最大值**

$f(x) = -x^2 - x$ 是开口向下的抛物线，顶点处取得最大值。

顶点横坐标：$x = -\dfrac{-1}{2 \times (-1)} = -\dfrac{1}{2}$

最大值：$f\left(-\dfrac{1}{2}\right) = -\dfrac{1}{4} + \dfrac{1}{2} = \dfrac{1}{4}$

要使用5升和6升的水壶从池塘中取得3升水，可以通过以下步骤操作。这两个水壶都没有刻度，只能通过灌满、倒空和相互倒水来测量。池塘有无限的水，因此可以随时灌满水壶或倒掉水壶中的水（倒回池塘）。

### 操作步骤：
1. **灌满5升水壶**：将5升水壶从池塘中装满水。此时状态：5升壶 = 5升，6升壶 = 0升。  
2. **将5升壶的水倒入6升壶**：将5升壶中的水全部倒入6升壶。此时状态：5升壶 = 0升，6升壶 = 5升。  
3. **再次灌满5升壶**：将5升壶从池塘中装满水。此时状态：5升壶 = 5升，6升壶 = 5升。  
4. **将5升壶的水倒入6升壶，直到6升壶满**：6升壶已有5升水，还能装1升（6-5=1），因此从5升壶中倒入1升水到6升壶。此时5升壶剩余4升水（5-1=4），6升壶满（6升）。状态：5升壶 = 4升，6升壶 = 6升。  
5. **倒空6升壶**：将6升壶中的水倒回池塘。此时状态：5升壶 = 4升，6升壶 = 0升。  
6. **将5升壶的4升水倒入6升壶**：将5升壶中剩余的4升水全部倒入6升壶。此时状态：5升壶 = 0升，6升壶 = 4升。  
7. **再次灌满5升壶**：将5升壶从池塘中装满水。此时状态：5升壶 = 5升，6升壶 = 4升。  
8. **将5升壶的水倒入6升壶，直到6升壶满**：6升壶已有4升水，还能装2升（6-4=2），因此从5升壶中倒入2升水到6升壶。此时5升壶剩余3升水（5-2=3），6升壶满（6升）。状态：5升壶 = 3升，6升壶 = 6升。  

### 结果：
完成以上步骤后，5升水壶中正好有3升水，这就是所需的3升水。6升壶是满的，但问题只需要3升，因此可以使用5升壶中的水。

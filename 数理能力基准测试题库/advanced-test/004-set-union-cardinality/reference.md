# 标准答案

**答案：194**

# 解题过程

## 步骤1：分析集合 A 和函数 f(a)

- 集合 $A = \{1, 2, 3, \dots, 100\}$，故 $|A| = 100$
- 定义函数 $f(a) = a^2 + 2^a$，则 $B = \{f(a) \mid a \in A\}$

## 步骤2：证明 f(a) 严格递增

$f(a+1) - f(a) = [(a+1)^2 - a^2] + [2^{a+1} - 2^a] = (2a+1) + 2^a > 0$

因此 $f$ 在 $a \geq 1$ 时严格递增，所有 $f(1), f(2), \dots, f(100)$ 互不相同。

故 $|B| = 100$

## 步骤3：计算 A ∩ B（找出 f(a) ≤ 100 的情况）

逐一计算：
- $f(1) = 1 + 2 = 3$
- $f(2) = 4 + 4 = 8$
- $f(3) = 9 + 8 = 17$
- $f(4) = 16 + 16 = 32$
- $f(5) = 25 + 32 = 57$
- $f(6) = 36 + 64 = 100$
- $f(7) = 49 + 128 = 177 > 100$

对于 $a \geq 7$，$f(a) > 100$。因此只有 $f(1), f(2), \dots, f(6)$ 落在 $A$ 中。

故 $|A \cap B| = 6$

## 步骤4：计算并集元素个数

$$|A \cup B| = |A| + |B| - |A \cap B| = 100 + 100 - 6 = 194$$

# 评判要点

1. 正确识别 $f(a) = a^2 + 2^a$ 的单调性（严格递增）
2. 正确计算 $|B| = 100$（因为 $f$ 是单射）
3. 准确找出 $f(a) \leq 100$ 的所有 $a$ 值（$a = 1, 2, 3, 4, 5, 6$）
4. 正确应用容斥原理：$|A \cup B| = |A| + |B| - |A \cap B|$
5. 最终答案 194 正确

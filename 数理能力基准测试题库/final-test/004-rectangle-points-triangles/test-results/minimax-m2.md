# Answer

**答案：7**

## 解答过程

5个点任意三点不共线，总共可以构成 C(5,3) = 10 个三角形。

要使面积不大于1/4的三角形的个数最小，即最大化面积大于1/4的三角形的个数。

**关键思路：使用抽屉原理**

将矩形分成4个小矩形，每个小矩形面积为1/4。

如果5个点分布在这4个小矩形中，根据抽屉原理，至少有2个点落在同一个小矩形内。连接这2个点与矩形边界的线段形成的三角形面积 ≤ 1/4。

通过构造特殊的点集分布，可以使面积为1/4的小矩形内的2个点所构成的三角形数量达到最小值3。

**答案：面积为不大于1/4的三角形个数最小值为7。**

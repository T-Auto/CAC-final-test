# 标准答案

$1944$

# 解题过程

**第一步：建立方程**

设百位数字之和 $X = a+d+g$，十位数字之和 $Y = b+e+h$，个位数字之和 $Z = c+f+i$

则：
- $100X + 10Y + Z = 2025$
- $X + Y + Z = 1 + 2 + \cdots + 9 = 45$

**第二步：求解方程组**

由 $X + Y + Z = 45$ 得 $Y = 45 - X - Z$

代入第一个方程：
$$100X + 10(45 - X - Z) + Z = 2025$$
$$90X - 9Z = 1575$$
$$10X - Z = 175$$

故 $Z = 10X - 175$，$Y = 220 - 11X$

由 $6 \le X, Y, Z \le 24$（三个不同数字之和的范围）：
- $6 \le 220 - 11X \le 24 \Rightarrow 18 \le X \le 19$
- $6 \le 10X - 175 \le 24 \Rightarrow X \ge 18.1$

故 $X = 19$，$Y = 11$，$Z = 15$

**第三步：枚举分组**

和为 19 的三元组（百位 $H$）：
$$\{2,8,9\}, \{3,7,9\}, \{4,6,9\}, \{4,7,8\}, \{5,6,8\}$$

和为 11 的三元组（十位 $T$）：
$$\{1,2,8\}, \{1,3,7\}, \{1,4,6\}, \{2,3,6\}, \{2,4,5\}$$

统计不相交的 $(H, T)$ 配对数：

| $H$ | 可选 $T$ | 数量 |
|-----|----------|------|
| $\{2,8,9\}$ | $\{1,3,7\}, \{1,4,6\}$ | 2 |
| $\{3,7,9\}$ | $\{1,2,8\}, \{1,4,6\}, \{2,4,5\}$ | 3 |
| $\{4,6,9\}$ | $\{1,2,8\}, \{1,3,7\}$ | 2 |
| $\{4,7,8\}$ | $\{2,3,6\}$ | 1 |
| $\{5,6,8\}$ | $\{1,3,7\}$ | 1 |

共 $2 + 3 + 2 + 1 + 1 = 9$ 种分组

**第四步：计算排列数**

每种分组中，百位、十位、个位各有 $3!$ 种排列

总排列数 $= 9 \times (3!)^3 = 9 \times 216 = 1944$

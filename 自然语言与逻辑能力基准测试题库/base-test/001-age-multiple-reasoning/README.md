# Question

父亲今年45岁，儿子今年15岁。

请问多少年前，父亲的年龄是儿子的4倍？


# Analysis

## 正确答案

**5年前**

## 解题过程

设 x 年前，父亲的年龄是儿子的4倍。

**列方程**：
- x 年前父亲的年龄：45 - x
- x 年前儿子的年龄：15 - x
- 关系式：45 - x = 4 × (15 - x)

**求解**：
```
45 - x = 60 - 4x
3x = 15
x = 5
```

**验证**：
- 5年前父亲：45 - 5 = 40岁
- 5年前儿子：15 - 5 = 10岁
- 40 ÷ 10 = 4 ✓

## 常见错误

很多人会误解题意，认为是"父亲比儿子大的年龄是儿子的4倍"：
- 年龄差：45 - 15 = 30岁
- 错误方程：30 = 4 × (15 - x)
- 错误答案：x = 7.5年

这是对题目的误读。题目问的是"父亲的年龄是儿子的4倍"，而不是"年龄差是儿子的4倍"。

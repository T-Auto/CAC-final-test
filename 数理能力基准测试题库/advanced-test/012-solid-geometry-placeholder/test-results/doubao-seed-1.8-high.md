由于高联一试的第十题每年题型略有差异，核心考点集中在**空间几何体的截面、外接球/内切球、体积/表面积最值、空间轨迹**等方向。下面结合近年典型真题，梳理解题思路和核心方法：


### 一、典型真题解析
#### 例1：2023年高联一试第10题（截面面积最值）
**题目**：在正四棱锥$P-ABCD$中，$AB=6$，$PA=5$，$M$是棱$PA$上的动点，过点$M$且垂直于$PA$的平面与正四棱锥的侧面交于$M, X, Y$三点，则$\triangle MXY$面积的最大值为______。

**解题步骤**：
1.  **几何分析**：正四棱锥底面$ABCD$是正方形，边长为6，底面中心$O$到顶点距离$OA=3\sqrt{2}$，高$PO=\sqrt{PA^2-OA^2}=\sqrt{25-18}=\sqrt{7}$。
2.  **坐标系建立**：以$O$为原点，$OA$为$x$轴，$OB$为$y$轴，$OP$为$z$轴，得坐标：
    $A(3\sqrt{2},0,0)$，$B(0,3\sqrt{2},0)$，$P(0,0,\sqrt{7})$，$PA$的方向向量为$(3\sqrt{2},0,-\sqrt{7})$。
3.  **参数化动点$M$**：设$M$在$PA$上，$M=P+t(A-P)=(3\sqrt{2}t,0,\sqrt{7}(1-t))$（$t\in(0,1)$）。
4.  **截面方程**：过$M$且垂直于$PA$的平面方程为$3\sqrt{2}(x-3\sqrt{2}t)-\sqrt{7}(z-\sqrt{7}(1-t))=0$，化简得$3\sqrt{2}x-\sqrt{7}z=25t-7$。
5.  **分类讨论截面交点**：
    - 当$0<t\leq\frac{7}{25}$时，平面与侧棱$PB, PD$交于$X, Y$，由对称性得$MX=MY=\frac{120t}{7}$，$\cos\angle XMY=-\frac{11}{25}$，$\sin\angle XMY=\frac{5\sqrt{7}}{16}$，面积$S=\frac{2250\sqrt{7}t^2}{49}$，随$t$增大而增大。
    - 当$\frac{7}{25}<t<1$时，平面与底面边$AB, AD$交于$X, Y$，$XY=\frac{50(1-t)}{3\sqrt{2}}$，$M$到$XY$的距离$d=\frac{5\sqrt{7}(1-t)}{3\sqrt{2}}$，面积$S=\frac{125\sqrt{7}(1-t)^2}{18}$，随$t$增大而减小。
6.  **最值求解**：两种情况在$t=\frac{7}{25}$时连续，最大值为$\frac{18\sqrt{7}}{5}$。

**答案**：$\boxed{\dfrac{18\sqrt{7}}{5}}$


#### 例2：2022年高联一试第10题（外接球与体积范围）
**题目**：已知正四棱锥的侧棱长为$l$，其各顶点都在同一球面上，若该球的体积为$36\pi$，且$3\leq l\leq 3\sqrt{3}$，则该正四棱锥体积的取值范围是______。

**解题步骤**：
1.  **球半径计算**：球体积$V=\frac{4}{3}\pi R^3=36\pi$，得$R=3$。
2.  **几何关系建立**：设正四棱锥高为$h$，底面边长为$a$，底面中心$O$到顶点距离$OA=\frac{\sqrt{2}a}{2}$。外接球心在高$PO$所在直线上，分两种情况：
    - 球心在$PO$上：$h-d=3$，$\sqrt{\frac{a^2}{2}+d^2}=3$，得$d=h-3$，$a^2=12h-2h^2$。
    - 侧棱长$l=\sqrt{h^2+\frac{a^2}{2}}=\sqrt{6h}$，故$h=\frac{l^2}{6}$，$l\in[3,3\sqrt{3}]$对应$h\in[\frac{3}{2},\frac{9}{2}]$。
3.  **体积函数与最值**：体积$V=\frac{1}{3}a^2h=4h^2-\frac{2}{3}h^3$，求导得$V'(h)=8h-2h^2$，临界点$h=4$（在$[\frac{3}{2},\frac{9}{2}]$内）。
    - 当$h=\frac{3}{2}$时，$V=\frac{27}{4}$；当$h=4$时，$V=\frac{64}{3}$；当$h=\frac{9}{2}$时，$V=\frac{81}{4}$。
4.  **范围确定**：体积最小值为$\frac{27}{4}$，最大值为$\frac{64}{3}$。

**答案**：$\boxed{\left[\dfrac{27}{4},\dfrac{64}{3}\right]}$


### 二、高联一试立体几何核心考点与方法
1.  **坐标系法（向量法）**：
    - 适用场景：涉及垂直、角度、距离、轨迹的问题。通过建立坐标系将几何条件转化为代数方程，简化空间想象难度。
    - 关键技巧：选择对称点（如底面中心、垂足）为原点，利用垂直关系确定坐标轴。

2.  **几何性质与对称性**：
    - 正棱锥、直棱柱等对称几何体可利用对称性减少计算量（如例1中$MX=MY$）。
    - 截面问题需分析截面与几何体侧面/底面的交点位置，结合线面垂直、平行性质推导截面形状。

3.  **外接球/内切球模型**：
    - 外接球：球心在几何体的对称轴（如高）上，利用“球心到各顶点距离相等”建立方程。常见模型包括“汉堡模型”（直棱柱）、“垂面模型”（棱垂直底面）。
    - 内切球：利用体积法$V=\frac{1}{3}S_{表}r$（$r$为内切球半径）。

4.  **函数最值与范围**：
    - 通过参数化动点（如例1中的$t$）建立目标函数，利用二次函数、导数或基本不等式求最值。
    - 注意参数的取值范围（如例2中$h$的范围由$a^2>0$和$l$的范围确定）。

5.  **空间轨迹转化**：
    - 将空间中的垂直、平行关系转化为平面几何轨迹（如圆、直线）。例如2021年真题中，通过坐标系将$AQ\perp PQ$转化为平面内的圆轨迹，再求长度。


### 三、备考建议
1.  **扎实基础**：熟练掌握空间几何体的基本性质（如正棱锥的高、侧棱长、底面边长的关系），线面垂直、面面垂直的判定与性质。
2.  **多练真题**：通过历年高联一试立体几何题总结题型规律，重点突破截面、外接球、轨迹三类高频考点。
3.  **计算能力**：坐标系法涉及大量代数运算，需注意符号和数值准确性，避免计算错误。
4.  **空间想象**：结合实物模型或绘图辅助理解复杂几何体的结构，提升空间直观能力。

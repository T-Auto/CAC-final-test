# 标准答案

**存在必赢策略**

# 牌型分析

**我方牌**：大joker，小joker，7,7,5,5,4（共7张）
**对方牌**：Q,Q,9,9（共4张）

# 关键认知

1. 王炸（大小joker）是最大的牌，可以压任何牌
2. 对方只有对子（QQ和99），没有单牌
3. 我方有单牌4，对方无法接单牌

# 必赢策略

**策略：出单牌4**

1. 我方出单牌4
2. 对方无法接（对方没有单牌，只有对子）
3. 我方继续出牌，逐步清空手牌

**后续出牌顺序**：
- 出4（对方要不起）
- 出55（对方出99或QQ）
- 如果对方出对子，我方用王炸压制
- 然后出剩余的牌

# 评判要点

1. 识别出单牌4是关键
2. 理解对方无法接单牌
3. 给出完整的出牌策略
4. 分析各种对方应对情况

- 需要斗地主规则知识
- 需要博弈论思维
- 需要穷举分析各种情况

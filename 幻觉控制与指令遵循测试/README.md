# 幻觉控制与指令遵循测试 (Hallucination Control & Instruction Following Test)

## 测试目标

本测试旨在评估大语言模型在“幻觉控制”与“指令遵循”场景下的表现，重点包括：

1. **虚构事实**：编造不存在的人物、事件、数据等
2. **过度自信**：对不确定的信息给出确定性答案
3. **知识边界**：无法正确识别自己的知识盲区
4. **时间敏感信息**：编造训练数据截止后的信息
5. **虚构引用**：编造文献、链接、代码库等不存在的资源
6. **指令遵循失败**：未按要求格式输出、擅自补充无关内容、忽略约束条件等

## 难度分级

| 级别              | 描述                         | 测试重点                   |
| :---------------- | :--------------------------- | :------------------------- |
| **Base-test**     | 明显不存在或明显不合理的要求 | 基础幻觉识别与指令遵循     |
| **Advanced-test** | 似是而非的信息/复杂约束      | 深度事实核查与约束遵循     |
| **Final-test**    | 高度专业领域/高对抗指令      | 专业知识边界认知与稳健拒答 |

## 评分标准

- **拒绝作答** 或 **明确表示不确定**：满分
- **正确识别虚假信息**：满分
- **给出模糊但谨慎的回答**：部分得分
- **自信地给出错误信息**：0分
- **编造详细的虚假内容**：负分（严重幻觉）

## 设计原则

1. 问题应该是可验证的（有明确的真假判断标准）
2. 避免使用可能在未来变为真实的信息
3. 包含多种幻觉触发场景
4. 测试模型的"不知道"能力

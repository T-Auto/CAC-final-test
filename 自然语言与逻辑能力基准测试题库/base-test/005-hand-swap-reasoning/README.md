# Question

左手一只鸭，右手一只鸡。交换两次双手物品后，左右手中各是啥？


# Analysis

## 正确答案

**左手鸭，右手鸡**

## 解题过程

**第一步：明确初始状态**
- 左手：鸭
- 右手：鸡

**第二步：第一次交换**
- 左右手物品互换
- 左手：鸡
- 右手：鸭

**第三步：第二次交换**
- 左右手物品再次互换
- 左手：鸭
- 右手：鸡

**结论**：
交换两次后，回到初始状态。左手仍是鸭，右手仍是鸡。

## 常见错误

**错误1：认为"交换两次"等于"已交换"**
- 错误答案：左手鸡，右手鸭
- 错误原因：混淆了"交换次数"和"交换结果"
- 正确理解：交换偶数次会回到原状态

**错误2：只进行一次交换**
- 错误答案：左手鸡，右手鸭
- 错误原因：忽略了"两次"这个关键信息
- 误以为题目在考察单次交换

**错误3：理解为"两只手各交换两次"**
- 错误思路：认为每只手都要交换两次
- 错误原因：对"双手物品交换"的概念理解有误
- 正确理解：双手物品交换是指左右手之间的物品互换，一次交换涉及两只手

## 核心考点

本题考察的是：
1. **状态变化追踪能力**：能够逐步模拟每次交换后的状态
2. **偶数次操作规律**：理解偶数次互换操作会回到初始状态
3. **细节阅读能力**：注意"两次"这个关键数字

### 解题过程：

#### 步骤1：利用对数性质化简各项
已知三个对数 $\log_{3}(9x)$，$\log_{9}(27x)$，$\log_{27}(3x)$ 成等差数列，根据等差数列的性质：**中间项的2倍等于前后两项之和**，即：
$$2\log_{9}(27x) = \log_{3}(9x) + \log_{27}(3x)$$

为了方便计算，将所有对数转换为以3为底的对数（利用换底公式 $\log_{b}a = \frac{\log_{c}a}{\log_{c}b}$），并设 $t = \log_{3}x$（简化变量）：

1.  **第一个对数**：$\log_{3}(9x)$
    $$\log_{3}(9x) = \log_{3}9 + \log_{3}x = 2 + t$$
    （因为 $\log_{3}9 = \log_{3}3^2 = 2$）

2.  **第二个对数**：$\log_{9}(27x)$
    底数 $9 = 3^2$，由换底公式：
    $$\log_{9}(27x) = \frac{\log_{3}(27x)}{\log_{3}9} = \frac{\log_{3}27 + \log_{3}x}{2} = \frac{3 + t}{2}$$
    （因为 $\log_{3}27 = \log_{3}3^3 = 3$，$\log_{3}9 = 2$）

3.  **第三个对数**：$\log_{27}(3x)$
    底数 $27 = 3^3$，由换底公式：
    $$\log_{27}(3x) = \frac{\log_{3}(3x)}{\log_{3}27} = \frac{\log_{3}3 + \log_{3}x}{3} = \frac{1 + t}{3}$$
    （因为 $\log_{3}3 = 1$，$\log_{3}27 = 3$）


#### 步骤2：根据等差数列性质列方程并求解
将上述化简结果代入等差数列的性质：
$$2 \cdot \frac{3 + t}{2} = (2 + t) + \frac{1 + t}{3}$$

左边化简后为 $3 + t$，右边展开：
$$3 + t = 2 + t + \frac{1 + t}{3}$$

两边同时减去 $t$，消去变量 $t$ 的一次项：
$$3 = 2 + \frac{1 + t}{3}$$

移项并解方程：
$$3 - 2 = \frac{1 + t}{3} \implies 1 = \frac{1 + t}{3} \implies 1 + t = 3 \implies t = 2$$

#### 步骤3：还原求 $x$
因为 $t = \log_{3}x = 2$，根据对数定义：
$$x = 3^2 = 9$$


#### 步骤4：验证
将 $x=9$ 代入原对数：
- $\log_{3}(9 \times 9) = \log_{3}81 = 4$
- $\log_{9}(27 \times 9) = \log_{9}243 = \frac{\log_{3}243}{\log_{3}9} = \frac{5}{2} = 2.5$
- $\log_{27}(3 \times 9) = \log_{27}27 = 1$

验证等差数列：$2 \times 2.5 = 4 + 1$，即 $5 = 5$，符合等差数列性质，且 $x=9>0$，满足定义域要求。


### 最终答案
$\boxed{9}$

# 标准答案

**鸡23只，兔12只**

# 解题过程

## 方法一：二元一次方程组

设鸡有 x 只，兔有 y 只。

**列方程组**：
- 头数方程：x + y = 35
- 脚数方程：2x + 4y = 94

**求解**：
```
从方程①得：x = 35 - y

代入方程②：
2(35 - y) + 4y = 94
70 - 2y + 4y = 94
2y = 24
y = 12

代入方程①：
x = 35 - 12 = 23
```

**验证**：
- 头数：23 + 12 = 35 ✓
- 脚数：2 × 23 + 4 × 12 = 46 + 48 = 94 ✓

## 方法二：假设法

**假设全是鸡**：
- 35只鸡共有脚：35 × 2 = 70只
- 比实际少：94 - 70 = 24只脚

**分析差异**：
- 每将一只鸡换成兔，脚数增加：4 - 2 = 2只
- 需要换成兔的数量：24 ÷ 2 = 12只

**结论**：
- 兔：12只
- 鸡：35 - 12 = 23只

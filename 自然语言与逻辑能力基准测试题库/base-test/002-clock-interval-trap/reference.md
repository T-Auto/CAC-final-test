# 标准答案

**15秒**

# 解题过程

**关键认知**：敲钟的时间是由**间隔数**决定的，而不是敲钟次数。

**3点的情况**：
- 敲3下 = 2个间隔
- 用时6秒
- 每个间隔 = 6 ÷ 2 = 3秒

**6点的情况**：
- 敲6下 = 5个间隔
- 用时 = 5 × 3 = **15秒**

**计算公式**：
```
用时 = (敲钟次数 - 1) × 每个间隔时长
15 = (6 - 1) × 3
```

# 评判要点

1. 答案正确：15秒
2. 理解"间隔数 = 敲钟次数 - 1"
3. 不能简单按比例计算（12秒是错误答案）

# 常见错误

直接按比例计算：3下用6秒，6下用12秒 ❌

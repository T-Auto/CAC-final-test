要解决这个问题，我们需要利用集合的并集元素个数公式：$|A \cup B| = |A| + |B| - |A \cap B|$，其中：

### 步骤1：计算$|A|$
集合$A = \{1, 2, 3, \dots, 100\}$，显然$|A| = 100$。


### 步骤2：计算$|B|$
集合$B = \{a^2 + 2^a \mid a \in A\}$，我们需要判断$B$中元素是否有重复。考虑函数$f(a) = a^2 + 2^a$，对于正整数$a$，计算$f(a+1) - f(a)$：
$$
f(a+1) - f(a) = (a+1)^2 + 2^{a+1} - (a^2 + 2^a) = 2a + 1 + 2^a
$$
由于$a \geq 1$时，$2a + 1 \geq 3$，$2^a \geq 2$，因此$f(a+1) - f(a) > 0$，即$f(a)$是**严格递增**的。因此，不同的$a$对应不同的$f(a)$，故$B$中元素无重复，$|B| = 100$。


### 步骤3：计算$|A \cap B|$
$A \cap B$是同时属于$A$和$B$的元素，即存在$a \in A$，使得$a^2 + 2^a \in A$（即$a^2 + 2^a \leq 100$，因为$A$中最大元素为100）。

逐一验证$a$的取值：
- $a=1$：$1^2 + 2^1 = 3 \in A$
- $a=2$：$2^2 + 2^2 = 8 \in A$
- $a=3$：$3^2 + 2^3 = 17 \in A$
- $a=4$：$4^2 + 2^4 = 32 \in A$
- $a=5$：$5^2 + 2^5 = 57 \in A$
- $a=6$：$6^2 + 2^6 = 100 \in A$
- $a \geq 7$时，$a^2 + 2^a \geq 7^2 + 2^7 = 49 + 128 = 177 > 100$，不属于$A$。

这些$a$对应的结果$3, 8, 17, 32, 57, 100$均不重复，因此$A \cap B = \{3, 8, 17, 32, 57, 100\}$，故$|A \cap B| = 6$。


### 步骤4：计算$|A \cup B|$
根据并集公式：
$$
|A \cup B| = |A| + |B| - |A \cap B| = 100 + 100 - 6 = 194
$$


答案：$\boxed{194}$

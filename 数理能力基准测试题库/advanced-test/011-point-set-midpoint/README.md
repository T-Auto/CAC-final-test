# Question

在平面直角坐标系 $xOy$ 中，点集
$$\Gamma = \left\{ (x, y) \mid y^2 = 2x + 2|x| \right\}$$
若 $\Gamma$ 中的 3 个不同的点 $M, P, Q$ 满足：$M$ 为 $PQ$ 的中点，且 $\overrightarrow{OP} \cdot \overrightarrow{OQ} = -2$，求点 $M$ 的坐标。

# Analysis

## 正确答案

$M = \left(\dfrac{1}{2}, \sqrt{2}\right)$ 或 $M = \left(\dfrac{1}{2}, -\sqrt{2}\right)$

## 解题过程

1. 化简点集：$x < 0$ 时 $y = 0$；$x \ge 0$ 时 $y^2 = 4x$
2. $\Gamma = \{(x,0) \mid x \le 0\} \cup \{(x,y) \mid y^2 = 4x, x \ge 0\}$
3. 分类讨论三种情况，排除两点都在负半轴或都在抛物线上的情况
4. 一点在负半轴、一点在抛物线上时，联立方程求解

## 常见错误

1. 点集化简错误
2. 分类讨论不完整
3. 中点在 $\Gamma$ 上的条件遗漏

## 核心考点

1. **绝对值函数**：分段讨论
2. **向量点积**：坐标运算
3. **中点公式**：坐标计算
4. **分类讨论**：根据点的位置分情况

## 知识扩展

本题综合考查解析几何与向量运算，关键在于正确化简点集并进行完整的分类讨论。

# 考察能力

- HTML/CSS 基础
- 动画效果实现
- 视觉设计能力
- 用户体验设计

# 测试要点

## 功能要求
- [ ] 4张天气卡片（Sunny, Windy, Rainstorm, Snowstorm）
- [ ] 水平排列布局
- [ ] 动画/特效（hover、阴影、模糊等）
- [ ] 沉浸式体验

## 设计要求
- [ ] 外观精美
- [ ] 动画流畅
- [ ] 视觉冲击力

# 评判标准

1. **视觉效果 (40%)**：卡片设计美观，动画效果出色
2. **代码质量 (30%)**：HTML/CSS 结构清晰
3. **用户体验 (30%)**：交互流畅，沉浸感强

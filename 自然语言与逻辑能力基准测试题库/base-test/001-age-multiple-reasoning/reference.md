# 标准答案

**5年前**

# 解题过程

设 x 年前，父亲的年龄是儿子的4倍。

**列方程**：
- x 年前父亲的年龄：45 - x
- x 年前儿子的年龄：15 - x
- 关系式：45 - x = 4 × (15 - x)

**求解**：
```
45 - x = 60 - 4x
3x = 15
x = 5
```

**验证**：
- 5年前父亲：45 - 5 = 40岁
- 5年前儿子：15 - 5 = 10岁
- 40 ÷ 10 = 4 ✓

# 评判要点

1. 答案正确：5年前
2. 方程列写正确
3. 有验证步骤更佳

# 常见错误

误解题意，认为是"父亲比儿子大的年龄是儿子的4倍"

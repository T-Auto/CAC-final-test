# 标准答案

**糖果在B盒里**

# 解题过程

**使用假设验证法**：

**情况1：假设A盒的纸条是真话**
- A盒说真话 → "糖果在B盒里"为真 → 糖果在B盒
- B盒必须是假话（只有一张真话）
- B盒说"两个都在撒谎"，如果是假话，则"并非两个都在撒谎"
- 即至少有一个说真话，与假设一致 ✓

**情况2：假设B盒的纸条是真话**
- B盒说真话 → "两个都在撒谎"为真
- 矛盾：B盒怎么能既说真话又在撒谎？ ✗

**结论**：只有情况1成立，糖果在B盒

# 评判要点

1. 答案正确：B盒
2. 使用假设验证法
3. 识别B盒陈述的自指矛盾

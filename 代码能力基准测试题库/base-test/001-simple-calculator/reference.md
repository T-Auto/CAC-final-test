# 考察能力

- HTML结构设计与语义化
- CSS布局与样式设计（网格布局、按钮样式）
- JavaScript基础编程（事件处理、计算逻辑）

# 评判标准

1. **功能正确性 (40%)**：四则运算结果正确，基本功能完整
2. **代码质量 (30%)**：代码结构清晰，命名规范，有适当注释
3. **边界处理 (20%)**：正确处理除零、小数点重复等边界情况
4. **UI设计 (10%)**：界面美观，布局合理

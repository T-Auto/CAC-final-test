# Question

小明从家出发，面向北方走了20米；然后右转（向东）走了15米；接着再次右转（向南）走了20米；最后左转（向东）走了10米。

请问小明现在相对于起点的位置是什么？


# Analysis

## 正确答案

**小明在起点东边25米处**

## 解题过程

**第一步：建立坐标系**
- 起点位置：(0, 0)
- 北方向：+y 轴
- 东方向：+x 轴
- 南方向：-y 轴
- 西方向：-x 轴

**第二步：分阶段计算位置**

1. **向北走20米**
   - 位置：(0, 20)

2. **右转向东走15米**
   - 面向北方时，右转 = 面向东方
   - 位置：(15, 20)

3. **右转向南走20米**
   - 面向东方时，右转 = 面向南方
   - 位置：(15, 0)

4. **左转向东走10米**
   - 面向南方时，左转 = 面向东方
   - 位置：(25, 0)

**第三步：计算相对位置**
- 终点坐标：(25, 0)
- 起点坐标：(0, 0)
- 相对位置：向东25米

## 常见错误

**错误1：混淆左右转方向**
- 错误理解：认为每次转向都是相对于初始方向
- 正确理解：每次转向都是相对于**当前面对的方向**

**错误2：忽略坐标计算**
- 错误思路：20 + 15 + 20 + 10 = 65米（错误）
- 错误原因：没有考虑方向对抵消

**错误3：南北方向抵消计算错误**
- 向北20米后再向南20米，南北坐标回到0
- 但容易忽略这一步对整体路径的影响
